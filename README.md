# ECL Language highlighter for Atom

## Overview

ECL syntax highlighting in Atom. Based on the [VS Code package](https://github.com/GordonSmith/vscExt) for the same language.

## Setup

While on atom:
1. Click on `File`
2. `Settings`
3. `Packages`
4. Look for the package named `language-ecl`
5. Click on `Install`
